# SQL Bookstore Database Project

A relational database project for a bookstore system built using SQL.

## Tables
- **Books** - stores book details, genre, price, stock
- **Customer** - stores customer info and location
- **Orders** - links customers and books with order details

## Queries Included
- 11 Basic Queries (filter, sort, aggregate)
- 8 Advanced Queries (JOINs, GROUP BY, HAVING)

## Tools Used
PostgreSQL / pgAdmin

## Skills Demonstrated
DDL, DML, Joins, Aggregations, Foreign Keys
